PHASE 3.5 Sandbox Test Execution Only with /ralphtemplatetest (skipped if TESTINGOFF)
Tester Runs Sandbox Tests
  • Executes tests from /tmp/ralph-test-sandbox-SESSION_ID/
  • Tests target the Builder's actual implementation
  • Tests pass: promise stays valid, proceed to cleanup
  • Tests fail: promise REVOKED, Builder sent back to Phase 2
Sandbox Cleanup
  • trap "rm -rf /tmp/ralph-test-sandbox-*" EXIT
  • /tmp is ephemeral (per-session)
  • No test code pollutes project tree
