PHASE 2 Build & Iterate (Ralph Loop)
Builder (Test-Driven)
  • Writes tests first, expects failures at current stage
  • Implements iteratively, tests after each change
  • Logs: what failed, root cause, fix applied, result
  • Max N iterations before escalating
  • Writes brief retrospective each iteration
Challenger Reviews
  • Reviews after each milestone
  • Checks edge cases & security
  • Checks performance at scale
  • Can force redesign
